Pocket Piano v.01

  • Oh dear oh dear... now you can make terrible tunes with your HPC.
    Download this little handheld keyboard, and tap out your favourite
    ditties.Music quality depends on your HPC, and your tapping ability.

    Please note that this is a very, very basic system. The only menu option
    which does anything is "Music/Keyboard". Sticky Software will not
    be offering support for this program at this stage.


    Download the file below to your desktop PC, and then copy it to your HPC.
    Both SH3 and MIPs executables are supplied.

    You will need to create a directory called "Piano" and copy all the files into it.
    This is important -- the directory must be under "My HPC." No other locations
    will work. If you don't hear anything, then this is the reason why. Check it and
    please don't email me.

    PocketPiano for MIPS and SH3

    This program makes use of the WinCEv1 MFC libraries. You must have a file
    called MFCWCE.DLL present in the Windows/ directory on your HPC for
    the program to work. Some HPCs have this file included in ROM, others
