Deluxe v2.0

You want it..

What are the two things Palmsize PC owners want the most from their User Interface?

That's easy:

1. A way to shut down open applications.

Microsoft's guidelines for the Palmsize PC request that developers don't include any options to close currently running applications. The idea is to let Windows CE look after the memory management itself, leaving you to just to get on with using the computer. If you are finished using the Notetaker for example, you just start the next application and Notetake with vanish into the background. If you need it again, you launch it from the Start menu as before: and WindowsCE will move it to the front of the display.

In theory this is a great idea. In practice, many users don't feel right about it. We want to shut down the programs we no longer need. We even go the extent of opening the pop-up keyboard and entering CTRL and Q to do it "manually". We still want the control over our programs which the Handheld PC and desktop versions of Windows give us. We don't want to wait for WindowsCE to run out of memory and shut down the applications: we want to do it ourselves!

2. A way to quickly return to frequently used applications.

As you normally don't or can't shut down applications, the programs in the background are wasted: they sit there sucking up resources. To get to them, you have to open the Start menu and find and launch them again - what a waste of time! How much better if you could just jump to them with only a few taps on the screen. Quicker, easier, more logical and more consistant with the Microsoft ideal of keeping the applications open in the first place.

..Flipper does it!

Flipper is a small program which achieves both these much-needed tasks in one, unobtrusive taskbar icon.

Flipper Deluxe is a Task Manager, but one which is both easy to use and packed with features. Install it, and then launch it – you'll notice a new icon appears in the tooltray at the bottom of the screen.

Flipper is this easy to use:

Single-tap on the Flipper icon, and the currently open tasks in the background are shuffled around. Your current program (or desktop) flips to the background, and a previous program flips to the foreground. Keep single-tapping and all the open applications will appear, one at a time. A good hint is to assign Flipper to one of the user-definable buttons on the Palm-size PC. Then you only need to press the button to cycle all your open applications.

Double-tap on the icon, the currently open task (excluding the desktop) will be closed. No fuss, no hunting for the Task Manager. One click, and the current task is history. This is without a doubt, the easiest way yet to keep your applications and tasks in check.

And more!

Flipper has another feature. Assign Flipper to one of your Palmsize PC's user-defined buttons, and you can quickly flip between applications with a single click. Just press the button, the next application flips to the foreground.

To open the pop-up menu, tap and hold the stylus on the Flipper icon for slightly longer than usual. The menu will pop up. The first items in this menu are the currently running applications. To jump to them, just tap its entry in the menu.

The remaining options are as follows:

  • Kill current task
    This kills the current application. It's does the same thing as double-tapping the icon.
  • Kill all but current
    This kills all the other programs, other than the one you are currently using (and/or the desktop). This is a very fast way to quickly clear out memory.
  • Options...
    This brings up a new dialog, with extra information and options. (See later).
  • Run...
    This brings up the Windows CE "run" dialog, into which you can enter program or document names in order to launch applications manually.

 

Options Window

The options window provides you with a quick summary of the memory available on your Palm-size PC, and the current status of the batteries. There are two other options:

1. Autokill

When this item is checked, Flipper will periodically close all the but the current window. So if you are using your Palm-size PC for some time-intensive task, you won't have to worry about saving memory or closing unwanted applications. Flipper does it for you. Please try this option. While it mightn't suit everyone's style of working, it can be useful.

2. Startup Sound

This feature will trigger a sound (called "flipper.wav") when you power on your Palm-size PC. Use this to provide amusing sound effects. One catch is that the sound will only be triggered if you have a CF card attached to your Palm-size PC. You might want to switch off this option if it interferes with your one-touch voice recording, or alarm clock sounds.
