New Version 2.1 Now Available!

Midnight bug fixed and Flash Memory support

Now your handheld computer can tell you the time:
literally! Using sampled waveforms your Palmsize PC
or Handheld PC can read out the time to the nearest
few minutes - all with a female voice, complete with
Northern Irish accent. ;-)

If you want, there is nothing to stop you re-sampling the
waveforms with your own voice.

Special "fuzzy logic" means you won't hear "The time
is one three zero seven", but instead you'll be told that
the time is "just after five past one".

Assign this program to one of your user-definable
keys, and you can be told the time without even
having to hunt for your stylus.

Using Speaking clock with a
Palmsize PC and FLASH CARD!

This is now possible. When installing the software, you will
be asked the question "Install Speaking Clock in the default
directory?"

Select "no" at this point. You will then see a dialog asking you
to chose between Main Memory and Storage Card -- select
the card. The data will now be installed into the Flash
memory, in a directory called "Sticky Software Speaking Clock".
It is vital that you do NOT rename this directory, or the program
will fail to operate.

If you wish to install Speaking Clock in main memory, just
answer YES to the question regarding the default directory.

Note: if you move the directories around or rename them,
the program will fail to operate!
