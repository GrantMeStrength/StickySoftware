Installing new themes

Currently, there is no "theme manager" program, although this will probably change in the near future. This means you have to do some of the file dragging and copying yourself: but don't panic, it's not hard! The key is that the image files are stored in the directory called Windows/Temporary Internet Files on your Palm-size PC.

Copying files is easy. Sync your Palm-size and desktop in the usual way. Open up the Mobile Devices folder on your desktop, and navigate your way to the Windows/Temporary Internet Files folder.

It's best if you make a back-up copy of the original icon files in case you change your mind at some point. In the Windows/Temporary Internet Files folder, find and select the following files: barney.gif, and but?.gif (that is, anything which starts "but" and ends in ".gif", such as "but1.gif", "butX.gif" and so on. Copy these files to a folder on your desktop to keep them safe.

You can then unzip the new files which make up the theme. Keep them in a named folder on your desktop system so you don't lose them, or confuse them with the originals. Now drag and drop the new files from the desktop. You may be told there is no file convertor for these files: that's perfectly normal. You can check the box to prevent that message from happening again. You will then be told that you are going to replace some files. Again, that's file: you can select "Yes to All" and let them copy across.

Once this process has finished, you can see your new desktop by forcing a redraw by going to Start/Settings/Display and clicking on OK.

How to create your own Sticky Button Themes

Sticky Buttons uses the following image files:

  • barney.gif
    This is a GIF file of 200 pixels wide by 54 pixels high which forms the optional logo at the top of the Sticky Buttons display. Don't try to use anything flash such as GIF animation or transparency, it just won't work. Call the file "barney.gif" and copy it to the usual directory.
  • but1.gif - but8.gif
    The standard application icons. Each one is 50 by 50 pixels, and includes a one word description. Obviously you can leave out the text, but only if your icon make its purpose obvious. There can only be four icons in a row at the most. When all are selected they form two rows of four. The icons always appear in the same order: i.e. but4.gif is always the link for Notetaker
  • but9.gif
    The default icon used to display user-defined applications. This icon is 32 by 36 pixels in size, and doesn't include any text. The text (that is, the name of the program) is added automatically when it is displayed. Up to four of these images can appear on the screen, all will use but9.gf
  • butF.gif, butV.gif, butP.gif, butM.gif, butT.gif, butX.gif, butH.gif
    These seven are the smaller icon files which make up the optional utilities bar. They are gif files of 26 by 22 pixels, with no text. The names appear in a fixed order (look at the screenshots on the previous page to work it out). Either all these files are displayed, or none are.

Note: although it is possible to mess around with the sizes, the values above were chosen after careful experimentation. Also, the sizes are built-into the Active Desktop display, so even if you use a logo file of 240 by 200 pixels, it will be resized to be 200 by 54. This was done to speed up the redraw of the desktop, and create a more stable display. Trust us!

If you want a larger logo, then it's a simple matter to save your larger file using the name "dttitle.gif" into the Windows/Temporary Internet Files folder, and using Start/Settings/Display to select the Logo desktop component.

This flexibility built into Sticky Buttons makes it possible to create all manner of strange and wonderful desktop displays. We hope you have fun designing your ultimate user interface!
