Block


Well, we had an afternoon to spare, so we wrote another
simple but addictive game. Block is the sliding block puzzle
of old -- re-arrange the sliding tiles to their initial form.

As well as the numbers 1-15, this version contains five other
pictures, which vary in difficulty. An ideal way to spend
a meeting or tedious train journey!


Block for the Palmsize PC is free. All we ask is that
you do not store the file on another web site or place it
on CD-ROM -- if you want to provide a link, then please
link to www.sticky.co.uk.Thank you.
