Welcome to the easiest Task Launching program, ever!

People who've seen it ask "Is this the way Windows CE should have been?"

It's been a long time coming, but it's been worth the wait. No other Windows CE task launcher works in this way. Sticky Buttons actually places familiar buttons right on the Active Desktop. No matter what other applications you have installed, the icons will be there waiting to lauch the key Windows CE applications (Notetaker, Voice Recorder, Inbox, Calculator, Tasks, Channels, Contacts and Calendar) and up to four applications of your choice.

Additionally, a logo file, the date and a bar of essential utilities can be displayed: you choose.

 

Utilities

 

 

 

 

By default, Sticky Buttons provides instant, one-tap access to all the key applications built-into the Nino. Contacts, Calculator, Notetaker -- all just one tap away. No need to open the Start menu: just tap, and that's it. The user can select which of these icons to display, and also assign up to four more for their own favourite programs.

Using Sticky Buttons is easy, as the buttons are part of the Active Desktop. This means they can be turned on or off from the standard Windows CE Display properties window.

 

 

FAQ

Q. How is this different from other launch programs?

A. The key difference is that the launch icons are displayed on the Active Desktop, not on a secondary window. Tap on the desktop icon (in the bottom right of the taskbar) and your launch icons are available.

Q. How large is the configuration program?

A. The configuration program is between 140 and 180K in size. It can moved to a CF card if required, or even deleted if you are happy with your button configuration.

Q. What else does it do?

A.As well as completely changing the appearance of your active desktop, and greating simplifying the way in which you will use your Palmsize PC, Sticky Buttons also has a bank of utility icons for power, battery and task management.

Q. Can I choose which programs appear on the desktop?

A. Yes. You can select all, some or none of the eight key Windows CE applications, and up to four of your own favourite programs too. The icons for your own user-defined programs are created from the programs themselves (note: except on colour systems, where a default icon is used instead).

Q. Does it work in colour / monochrome?

Q. Can I use my other graphics?

A, Yes! And we'll soon be providing some free samples very soon for you to download. New logo files and button images are easy to create and install.

Q. Can I mix the Active Desktop display with other components?

A. Yes, Sticky Buttons appears as a standard desktop component, so you can switch on the other components, alter the order and so on. You could elect to use only the four user defined buttons for example, or only the utilties, and have plenty of space for other components such as appointments, tasks or a logo.

Q. How can I uninstall it completely?

A. You can use the standard Remove Programs utility. However, due to the way active desktop components are inserted into the system, this will still leave a few files lurking around. Follow this like to see how to purge your system totally.

Q. Does it work on a handheld PC / handheld PC Pro?

A. No, only Palmsize PCs. Bottom line: do not buy this program if your Wndows CE computer has an integral keyboard!
