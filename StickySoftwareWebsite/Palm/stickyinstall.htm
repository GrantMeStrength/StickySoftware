Installing Sticky Buttons

 

Sticky Buttons integrates itself tightly into the Windows CE system, and so installation can seem a little involved. Here's a step-by-step guide.

Please read through these instructions before proceding. If you have any questions, please email support@sticky.net at any time.

Installing
Sticky Buttons
step-by-step.		 1.Once you have downloaded the self-extracting archive, you should run it on your desktop computer. You will be asked to connect your Palmsize PC, and the programs and data will be copied over.

 
2. Now launch the StickyButtons configuration program, which you will find under Start/Programs/Accessories. This is the program which you use to select the icons which will appear on the desktop, including the programs linked to the user-defined icons. However, at this stage the program needs to be launched to create registry settings and ensure files are located in the correct location.

 

 

 

 

 
3. There is no need to alter any of the default settings at this stage. The important point is that the program has been started. All you have to do now is click on the OK button.
4. Now you need to make sure the Active Desktop Display is switched on. To do this, click on Start/Settings/Display to open the Display Control Panel applet.

 

 5. Make sure that Display Active Desktop is on, and you'll notice an extra item: Sticky Buttons. For the moment, make sure it is on and turn off the others. Click on OK to close the Display Properties window.
6. That's it! After a few seconds, the desktop on your Palmsize PC will be re-drawn, with your new launch icons. Now you can re-visit the StickyButtons configuration program and make any changes, such as adding your own icons. You do not need to visit the Display Settings window again, unless you need to add more items (e.g. tasks, messages) and/or change their order.
