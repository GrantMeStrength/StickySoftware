Removing Sticky Buttons

 

Sticky Buttons integrates itself tightly into the Windows CE system, and so removing it totally can be a little involved. Here's a step-by-step guide.

Please read through these instructions before proceding. If you have any questions, please email support@sticky.net at any time.

Removing
Sticky Buttons
step-by-step.		 1. If you have decided to remove Sticky Buttons, you can simply use the standard Remove Programs setting. However, we recommend you perform the following steps first:
2. Go to Settings/Display and switch off Sticky Buttons. Now, if you have moved it, use the arrrow keys to push it to the very end of the list. This is very important!
  3. From Start/Settings open the Taskbar window. Now switch on Autohide to cause the Taskbar to slip off the bottom of the screen.
4. Launch the StickyButtons configuration program. You'll see a new button, which was previously hidden by the Taskbar. Tap it, and the registry entiries and other files will be remove.
  5. Now use the standard Remove Programs dialog from Start/Settings to finish the process. There! All gone!
