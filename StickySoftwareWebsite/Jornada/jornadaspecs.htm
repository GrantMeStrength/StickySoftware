Jornada Technical Specifications

So, what is it?

General

  • The HP Jornada 820 is a "Jupiter" class Windows CE computer. It's sub-notebook sized, with a colour LCD display, keyboard and glide-pad. In common with other Windows CE devices, it has no hard drive or floppy disk drive, and stores all information in battery-backed or flash RAM. It is designed as a "companion" device for existing computers, and synchronises data with other computers by means of serial or IrDA connection.

Operating System

  • Windows CE, HPC Pro class.

Processor

  • StrongARM SA1100 running at 190MHz

Physical

  • Dimensions: 24.6 × 17.8 × 3.3 cm (9.7 × 7.0 × 1.3 in)
  • Weight: 1.1 kg (2.5 lbs) with batteries

Memory

  • Internal RAM: 16 MB
  • Internal ROM: 16 MB Burst Mode ROM Upgradeable Package
  • Additional 16Mb may be fitted internally, without using up expansion slots.
  • Compact Flash slot
  • PC Slot (PCMCIA slot) way also house CF memory using adaptor.

Screen

  • 21.0-cm (8.2-in) VGA CSTN screen
  • 640 by 480 x 256 colours on screen
  • 1024 x 768 x 256 colours on external monitor (with built-in VGA-OUT capability)

Sound

  • Audio speaker and microphone

Special Features

  • Near-full size (90%) Keyboard, suitable for touch-typing
  • Glide-pad (instead of touch screen)
  • Display can be sent to external monitor (ALL programs, not just Pocket PowerPoint)
  • Integral software modem 56Kbps, v90

Expansion and Ports

  • CF and PC Slot (PCMCIA) slots
  • Single USB port
  • Video out
  • Modem outout
  • Serial output
  • IrDA port

Power

  • HP Jornada Standard Battery (rechargeable Lithium-ion battery pack)
  • HP Jornada power management software
  • Up to 10 hours of battery life with standard battery, 15 hours with optional extended battery
  • Two 3V CR2032 coin-cell backup batteries

Built-in Software

  • Usual suite of Windows CE Pro software, including Pocket Word, Excel, Access, and Internet Explorer.
  • HP Utilities for setting up ISP account, setting up user-defined keys, backing up data to CF, a special combined contacts, to-do and calendar view and a display perferences program.

Supplied Software

  • Java Virtual Machine from Microsoft
  • TrueSync (allows synchronisation with REX/REX Pro)
  • Omnisolve calculator
  • bFAX, bFind and others.
