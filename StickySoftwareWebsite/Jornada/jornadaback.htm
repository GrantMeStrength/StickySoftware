Wallpaper Images

It's very easy to personalise your Jornada by altering the default colours, or re-arranging the desktop icons. However, the most dramatic changes happen when you alter the wallpaper -- the image which is displayed on the desktop.

The image can be either a small pattern repeated (tiled) or a full-screen image, of 640 by 480 pixels. To install a backdrop, simply right-click on an empty part of the desktop, and select Properties. Then click in the Browse button to select a new image.

When creating an image, try to reduce the number of colours as much as possible before copying the image over. The Jornada palette contains up to 256 colours, but the range is still quite limited. Stick to a few colours, and use dithering to improve the image quality. Save the image as a BMP, and copy it to your Jornada.

Here is a selection of image which you can download and use as your own desktop wallpaper. These images have a strip on the left hand side which is colour matched to the colour of the text behind the icons.Arrange your icons in this strip for best results.
