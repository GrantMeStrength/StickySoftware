More Wallpaper Tips

The colour used as a background to the text of icons on the desktop can be changed from blue/green, meaning you don't have to have that colour on your wallpaper image. Go to Display Properties/Appearance/ and select the item: box element as Desktop. Alter this colour to suit your wallpaper image.

Settings Extras

The HP Settings utility has a few little extra features which you may have missed. This is the utility which is launched by pressing the right-most of the quick launch keys. You might have noticed that some of the icons have a border around them: for example, the Speaker icon which adjusts the volume. Clicking on these icons reveals they are actually buttons.

  • The Speaker icon launches Volume and Sounds applet
  • The Battery icon launches Power applet
  • The Memory icon launches the System/Memory applet

Noisey Modem?

If you find the Jornada's built-in modem is a little too noisey (have you been discovered dialing in for your mail at 2am?) then go to the Device Properties of the modem, select the Call Options tab and in the box marked Extra Settings, enter "M0" (that's a letter M, and a zero, without the quotation marks).

This setting will force the modem to be silent.

Don't forget the button!

That white blob at the top of the case? Wondered what it does? Press it: it lights up! Amazing... No, wait, there's more. If your Jornada is playing an alarm, press it to cancel. No need to open up the case, and so an ideal way to shut it off if you are using the computer as an alarm clock. (Thanks to H. for the tip)

More Keyboard shortcuts

Here are some more "in the know" key presses to speed up using your Jornada.

  • Windows and '-' will launch the PC Link communications utility
  • Alt and Tab will start the Task Manager
  • Within Pocket Word, hold down Alt, type a number (e.g. 4 and 2), release Alt.
    You can enter a huge range of characters in this way.
  • Alt and Esc will jump to the next open Task
  • Hold down S during a reset to start the self-diagnostic utility.

 

Creating a desktop wallpaper pattern

I've been experimenting with desktop wallpaper, and my favourite image is displayed below. This is a full 640 by 480 BMP image, taken from a scanned photograph and dithered to 256 colours in Photoshop. The strip on the left was included specifically to give the desktop icons somewhere to sit, and then colour matched to the background icon text (R:0 B:255 G:255). As you can see, the effect is quite pleasing -- not least because of my lovely wife on her motorbike! Image shimmering is present, but only very slightly. As the image has already been processed down to 256 colours, it's not very large and it doesn't seem to slow down the Jornada any.

Copying the image across was easy with the drag-and-drop desktop services. On the Jornada, selecting the Display Properties meant it was a simple matter to use the new image as wallpaper.

Use the Shortcut Keys

There are many shortcuts which can be achieved using the Windows key (there had to be a reason for it!). Some of the more useful ones are:

  • Windows on its own brings up the Start menu
  • Windows and H brings up Help
  • Windows and E starts Windows Explorer
  • Windows and C brings up the control panel
  • Windows and R bring up the RUN dialog
  • Windows and I brings up Mouse (Input) Properties
  • Windows and K brings up Keyboard Properties
  • Windows and M minimises all windows and returns you to the desktop (very handy)

Remember also these work in all programs and dialogs:

  • Control and C for Copy
  • Control and V for Paste
  • Control and X for Cut

Finally:

  • ESC will cancel most dialog windows. (thanks to Joe for the tip)

bFind

The bFind utility included with the Jornada can be a very quick way to locate often used files. Best of all, it can live in the tooltray ready for action. Start it running (Start/Programs/Accessories) and then select File/Use Try Icon. This will cause bFind to leave a little icon in the tooltray,sitting beside the time display. When you next need to find a file, double-click on the icon and up pops the window. Select the type of file you want to look for from the list on the left, click the "Include Subfolders" and then type the word or phrase you need in the Look for: box. Frequently used files can be turned into Shortcuts for ease of access.

Use a USB Mouse!

If you are using the Jornada at a desk, use the USB socket to provide your computer with a mouse. The MS Intellimouse V3.0 USB and Kensington Mouse-in-a-box mouse (another standard-sized mouse, available from dealers such as CompUSA) both work very well. Drivers for the mouse are built into the Jornada's ROM, so it's simply a matter of plug-in-and-go. One catch is that you cannot use the scroll wheel on the mouse, only the ordinary left/right buttons.

Touchpad Interfering with Typing

The touchpad is ultra-senstive, and if you brush it while typing it can move the cursor and mess up your document. HP thought of this, and included an option called "Disable touch pad when typing" in the "HP touch pad properties". Select this, and the touchpad is temporarily stunned for a fraction of a second after you type.

 

Multiple Selection

My Jornada was driving me nuts because I couldn't seem to multiple select items in the Inbox program. When I held down CTRL to select several items, nothing happened.

I found that this didn't happen when using the USB mouse, and I then realised it was because I had turned on the setting "Disable touch pad when typing" in the "HP touch pad properties". I find this option totally essential by the way.

To select multiple items with this option on, and using the touchpad, click on the first item as normal and then press CTRL. Instead of clicking on the second item, press the space bar. This seems to free up the pad motion, and you can now continue to hold down CTRL, move the cursor to the next item and click to select.

Moving between open documents

If you often have several Word documents open at once (making it easy to copy and paste information between them for example) you might find these hints useful.

Pressing the Word shortcut key only opens one instance of Word: so if you are in another window, and want to get back to Word, press it again. If you want to open a second document, hold down CTRL as you press the Word shortcut key.

While multiple Word documents are open, shuffle between them by pressing the Word shortcut button, or by using CTRL and "." and CTRL and ",".

USB Peripherals

So far the only USB peripherals supported by the Jornada are the MS Intellimouse V3.0 USB and the Kensington Mouse-in-a-box mouse. Keyboards, cameras, ZIP drives and so forth are not supported: yet.
