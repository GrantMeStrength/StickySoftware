Jornada Software

The amount of software for the Jornada 820 is increasing every day. Microsoft have finally finished the HPC Pro SDK, so expect even more over the weeks and months to come...

PowerPoint Converter NEW
Convert files on the Jornada itself, rather than through services.. This is provided by HP themselves.

TankZone NEW-ish
A 3D wireframe graphics shoot-em-up for the Jornada 820.

PowerToys NEW-ish
Get it now: that's all you need to know. Along with the PlusPack (below), this is the finest collection of utilities you can find for your Jornada 820, no question.

Plus Pack NEW-ish
A double-whammy from Microsoft, as they unleash a fabulous collection of freebies for your 820. The essential download.

Filegram
Send files over a serial link using XModem. Very useful for copying files to a PC which doesn't have CE Services installed.

JZip
Home of a useful utlity for un-archiving files compressed in Zip format. Also, a project planning utility and information on Java.

Pocket Streets 3
Excellent news! Microsoft has released Pocket Streets, the wonderful mapping software. Bad news in that older maps aren't compatible with this release, but at least it exists. Hurrah! Sample maps (including London <gasp> )are also available from the link above.

PowerView
A useful little battery monitor from Sticky Software

Nuance
The beginnings of what could be real alternative to Pocket Word for text editing.

VNC Viewer
An amazing program which let's you view your desktop PC's screen and interact with it over networks. Here's a free port of the viewer utilitity. You'll need the free server too, from the VNC homepage.

Footprint Software
Plan projects better with this new Pocket Outliner tool for your Jornada 820.

CompanionLink
Synchronises Contact, Calendar and To-Do information from ACT!, GoldMine, Lotus Organizer and Microsoft Outlook to your Jornada.

AnyWare Consulting
Looking for Jornada software to keep track of your cash? Try Pocket Finance 2.0, avialable from LandWare.

Cambridge Computers
A new telnet/serial terminal program for the Jornada.

bSquare
Windows CE veterans bSquare have a heap of very high-quality software for the Jornada, including the new bUseful pack of goodies. Make sure to visit their site. Nuff said?

JETCET
"The ability to print COLOR documents from a Windows CE-based Handheld PC has arrived via JETCET PRINT."

XTCE
Fancy being able to run existing DOS programs on your Jornada? Leisure Suit Larry for example? Or even Flight Sim? Or Word Perfect? The release version of this amazing program is now available for the Jornada. Incredible: relive the Days of Dos.

Ilium Software

Ilium have just announced versions of eWallet, ListPro, Keep Track, Recordian and eSportsCard! I've used the pre-release version of the Jornada port of eWallet, and it is an excellent program, making it easy to carry vital information around with you in a format which is quick to enter and read. Defintely recommended as one of the programs you will use most.

The Good Book
Not one, but two HPC Pro versions of The Bible to choose from, at http://www.laridian.com/ and http://www.slsoftware.com/index.shtml. Remember too that the text of the Bible is itself freely available if you don't need a fancy reader.

XAudio
MP3 is an audio compression format which can shrink files to a 30th of their original size. A Jornada compatible version is now available. Carry your favourite music with you on a CF card..

Developer One

Developer One is responsible for AccessPanel: one of the most useful utilities for the Windows CE range. I've tested the pre-release version of this program on the Jornada, and found it just as useful as it is on other platforms (i.e. very). Definitely worth checking out, as it can speed up almost all aspects of your daily Jornada use.

GrundleSoftware
Grundle has a long list of Windows CE software, and have ported almost the entire range to the HPC Pro, including the Jornada. The Registry Editor alone is an essential program if you intend on exploring your new computer's internal operations in any detail. Click here for their Beta Software page, which is full of Jornada versions.

mov software
Looking for an IRC client to chat on-line? Here's a BETA specially for the Jornada's ARM processor.

NSBasic
If you want to write your own programs on your Jornada, here's a professional development system based on good old BASIC. Enter and run programs entirely on your Jornada, complete with GUI support.

DeClock
Use your Jornada as an expensive desktop clock... Nice wallpaper feature.

Moose Software
Like card and board games? Here's some more to play with on your Jornada.

Rukson
Dedicated CE developers, Rukson, have FTP, News and Telnet clients available for the Jornada.Now you can read Usenet news no matter where you are.

Phatware
Notes software for all Windows CE machines: this is how you would synchronize notes with Outlook, for example.

Word Count
At last! A free word count for the Jornada. It's been driving me crazy trying to guess the length of articles...

Xoanan
Action games for your Jornada -- throw away the cards and play Martians and Sabotage!

MidiPlay
If you happen to have an external MIDI instrument, you can now replay songs from your Jornada!

HandMail
CE software for accessing AOL mail. Hassle them for a Jornada version!

ZTerm
A fully-featured terminal emulator for the Jornada. Due out soon.

Microsoft
The big "M" produces several items of interest to Jornada owners. First of all, the Mobile Channels update allows you to view web pages off-line, and with the Jornada's wonderful colour display they look great. Secondly, MS are resposible for the Entertainment Pack which contains many games which run on the Jornada. Worth checking out. Finally, for the moment at least, MS have produced an update to the security aspects of Pocket Internet Explorer, by providing 128 bit encryption: useful for home banking, for example. This update is only available to US/Canada based Jornada users (as the US government classes such encryption as military hardware, and we Europeans are obviously all terrorists and drug dealers) and it can be downloaded from the MS site. Here's hoping the PC Pro PowerToys will be out soon...
