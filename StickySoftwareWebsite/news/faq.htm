Sticky FAQ

Questions you probably didn't
want to know the answers to...
Q. Who are you guys?

A. Oh, you know, just some brain cells somewhere.

Q. Why Sticky?

A. Sticky was the name of a band (don't ask, you don't want to
hear anything) and then came the Web site. Once the handheld
PC came along, wow: we just had to develop for it. So we
taught ourselves Visual C++ and the rest is history.

Q. How come your software is free?

A. Not all of it is! Pocket Universe is a commercial product, which
can be bought from the Palmsize PC Store, as can FlipperPro.
Pocket Universe took over a year to develop, and we think the price
is very low considering that many PsPC games cost even more!.
Also, it's an amazing program and we would like some reward for our hard
work: mostly so we can buy a nice Meade LX200. And a Honda Fireblade.

TankZone is not free. It's charity-ware. We're serious: if you use it,
make a donation.It's the least you can do.

Other software is free because frankly it's not worth our while trying
to collect payment for it. If we tried to charge, say, five dollars for
HangMan, then we would probably lose out on each copy we sold.
And people would complain that it didn't interface directly with the
Project Gutenberg site.

If you want to send us money however, don't let that stop you.
Send email to iwantyoutohaveanewtelescope@sticky.co.uk and
we'll gladly give you our bank details.

Q. Aren't you worried that giving away software for free will
devalue software development as a whole?

A. Um. No. More worried about not getting enough exercise.

Q. How come I don't see your software on any of the usual
Windows CE sites?

A. Sticky Software currently does not agree with Web sites which
provide links directly to EXE or ZIP files, or who store files locally
on their own web site. Some sites even demand you "register" with
them to download other people's free software! Madness!
Other reasons include:

1. We want to you visit our web site for yourself. That way you can
check up the background information on the software, read the
updated support and FAQ news and also find out about the other
programs we have available. If you simply click a link and
download the software, you are very unlikely to visit this site.

2. If another Web site stores our software, we have no control. We
cannot ensure that you are using the latest version. If you download
it from here, you know it's the most current version available.

3. To help us plan, we need to know which programs are the most
popular. We can only tell this if the software is downloaded from
our site. We do not use Cookies, or store the names or details of
anyone who visits this site -- but we can tell how many downloads
were made.

4. Some software can ONLY be downloaded and installed from a
particular location, due to the way it works. This isn't our doing, it's
the way Mobile Channels work. Sadly, some large web directories
are so short-sighted that they cannot appreciate this fact.

So if you don't see TankZone (say) on CE Monster, then you know
the Website in question has decided that it knows best for you and
has refused to provide a link to Sticky Software. We think this sucks,
but hey, if you're reading this you got here. We hope you find
something of interest.
