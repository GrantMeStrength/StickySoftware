Physical Features

Which processor do you use?

    We selected the industry leading StrongARM SA1100, a 32-bit RISC processor (190MHz), because it is specially designed to maximize system performance while using minimal power, which is critical for a mobile product.

    How large is the Keyboard? Can I touch type on it ?

    • The keyboard is about 90% of a full-sized PC keyboard and is touch typeable. It also has an integrated touch pad and 3mm key travel. This makes the keyboard very comfortable and convenient to use.

      Go Back

    Why did HP choose to use a touch pad instead of a touch screen ?

    • Targeted for touch type users, HP opted to use a touch pad for the HP Jornada 820 because it is easier for touch type users to type and navigate around the display without disruptions. Touch screen users, however, are forced to disrupt their work (pick up and put down pen) each time they need to navigate around the display.
    • A large touch screen not only adds weight to the product but is more vulnerable to damage which makes the product less reliable than a product using a touch pad.
    • A large touch screen also obscures the display back lighting by about 20% - users have to turn up the display brightness to compensate which significantly reduces battery life.

    Power

    What is the battery life?

    Battery life will depend on display brightness, application usage, battery condition, power management settings and other factors. With continuous (non-stop) typing in Pocket Word and using default product settings, the standard battery (included in product) should last 10 hours, and the extended battery 15 hours.

    How long does it take to charge & re-charge the battery?

    • Standard battery: about 3 hours for a full charge (approx. 2 hours for an 80% charge), with the product turned off.
    • Extended battery: about 4.5 hours for a full charge (approx. 3 hours for an 80% charge), with the product turned off.

    How can I maximize battery life?

    • Set the brightness to the lowest comfortable setting and increase contrast instead.
    • Set the auto-dim delay option to the lowest comfortable time. This will dim the screen to conserve power yet allow information to be seen when the user is not using the unit.
    • Set the auto-suspend to the lowest comfortable time.
    • Set the volume to the lowest comfortable setting
    • Use the AC adapter as far as possible when using the modem (for communications), VGA-out (for presentations), USB or serial ports (eg. for connecting to a PC). These operations draw higher power.
    • Reading from and writing to a memory card uses more power than using internal memory.

      Go Back

    How long can I use a modem while on battery power?

    The answer depends highly on modem type, speed (higher speeds require more power), chipset and even connector (internal or cable). With the HP Jornada 820 internal modem and typical usage and settings, battery life is estimated to be reduced by about only 20%

    No, the extended battery is also a Lithium ion battery but with 50% more capacity. A big advantage is that the extended battery is physically the same size as the standard one, so it does not make the product bigger to carry around.

    Memory

    Can the memory  be expanded?

      Yes, in 2 ways:

      i) The HP Jornada 820 has 2 expansion slots: a PC Card, and CompactFlash card slots. Customers can plug in external storage into these card slots to store more data, on top of the built-in internal memory in the product. They can use the external storage to backup their Pocket Outlook (Calendar, Contacts, Tasks) or all data - a HP special - and travel safely away from their desktop PC. They can also open Pocket Word or Pocket Excel data files off the external storage, and will therefore not be constrained by the internal memory size. Examples of external storage are Sandisk CompactFlash memory cards and Iomega Clik! Drive.

      The internal memory can be partitioned into storage memory (for storing data) and program memory (for running applications) and users can decide on how they want to partition this. Adding memory cards as described above can increase storage memory. If users need a lot of program memory, they can partition the internal memory to achieve this & use memory cards for additional storage.

      ii) If more internal memory is needed, users can also expand their internal memory with an optional 16MB RAM module that is user installable, and increase internal memory to 32MB.
